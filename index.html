Stellar Chatbot Specifications

Technical details and requirements for the Stellar Chatbot system

Overview

The Stellar chatbot is a lightweight, responsive web application that provides an interactive conversational experience using the gemma3:1b language model via the Ollama API. It features a clean, modern interface with dark/light mode support and real-time response streaming.

Functional Requirements
  • 1. Chat Interface
    • Real-time message display with user/bot differentiation
    • Responsive input area with send button
    • Automatic scrolling to newest messages
    • Message history preservation (last 10 messages)
  • 2. Theme Management
    • Toggle between light and dark modes
    • Persistent theme preference across sessions
    • Adaptive UI elements for both themes
  • 3. Response Processing
    • Markdown formatting for responses
    • Code block highlighting and copy functionality
    • Streaming response display
    • Loading indicators during processing
  • 4. System Integration
    • Ollama API communication
    • Error handling and user feedback
    • Request cancellation support

Technical Architecture
  • Frontend: HTML5, CSS3, JavaScript (no frameworks)
  • Backend Integration: Ollama API endpoint
  • Model: gemma3:1b with custom parameters
  • Data Format: JSON for API communication

Limitations
  • Context limited to ~2048 tokens
  • Requires local Ollama server running
  • Performance depends on local hardware
  • Basic error handling only
